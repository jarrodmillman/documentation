networkx.algorithms.bipartite.basic.bipartite_density¶

networkx.algorithms.bipartite.basic.bipartite_density(B, nodes)¶

Return density of bipartite graph B.

Parameters :

G : NetworkX graph

nodes: list or container :

Nodes in one set of the bipartite graph.
Returns :

d : float

The bipartite density

See also

bipartite_color

Examples

>>> G = nx.complete_bipartite_graph(3,2)
>>> X=set([0,1,2])
>>> nx.bipartite_density(G,X)
1.0
>>> Y=set([3,4])
>>> nx.bipartite_density(G,Y)
1.0
