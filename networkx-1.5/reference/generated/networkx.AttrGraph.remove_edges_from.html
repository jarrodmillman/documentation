networkx.AttrGraph.remove_edges_from¶

AttrGraph.remove_edges_from(ebunch)¶

Remove all edges specified in ebunch.

Parameters:

ebunch: list or container of edge tuples :

A container of edge 2-tuples (u,v) or edge 3-tuples(u,v,d) though d is ignored unless we are a multigraph.

See also

remove_edge
remove a single edge
MultiGraph.remove_edges_from
the MultiGraph allows for parallel edges to be removed

Notes

Will fail silently if the edge (u,v) is not in the graph.

Examples

>>> G=nx.path_graph(4)
>>> ebunch=[(1,2),(2,3)]
>>> G.remove_edges_from(ebunch)
