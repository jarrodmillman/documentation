diameter¶

diameter(G, e=None)[source]¶

Return the diameter of the graph G.

The diameter is the maximum eccentricity.

Parameters :

G : NetworkX graph

A graph

e : eccentricity dictionary, optional

A precomputed dictionary of eccentricities.
Returns :

d : integer

Diameter of graph
