networkx.write_gpickle¶

write_gpickle(G, path)¶

Write graph object in Python pickle format.

This will preserve Python objects used as nodes or edges.

G=nx.path_graph(4) nx.write_gpickle(G,”test.gpickle”)

See cPickle.
