networkx.MultiGraph.__len__¶

MultiGraph.__len__()¶

Return the number of nodes. Use the expression ‘len(G)’.

Returns:

nnodes : int

The number of nodes in the graph.

Examples

>>> G = nx.Graph()   # or DiGraph, MultiGraph, MultiDiGraph, etc
>>> G.add_path([0,1,2,3])
>>> print len(G)
4
