networkx.node_connected_component¶

node_connected_component(G, n)¶

Return nodes in connected components of graph containing node n.

Parameters:

G : NetworkX Graph

An undirected graph.

n : node label

A node in G
Returns:

comp : lists

A list of nodes in component of G containing node n.

connected_components
