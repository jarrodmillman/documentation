clustering¶

clustering(G, nodes=None, weight=None)[source]¶

Compute the clustering coefficient for nodes.

For unweighted graphs the clustering of each node is the fraction of possible triangles that exist, For each node find the fraction of possible triangles that exist,

For weighted graphs the clustering is defined as the geometric average of the subgraph edge weights [R143],

Parameters :

G : graph

nodes : container of nodes, optional (default=all nodes in G)

Compute clustering for nodes in this container.

weight : string or None, optional (default=None)

The edge attribute that holds the numerical value used as a weight. If None, then each edge has weight 1.
Returns :

Notes

Self loops are ignored.

References

[R143](1, 2) Generalizations of the clustering coefficient to weighted complex networks by J. Saramäki, M. Kivelä, J.-P. Onnela, K. Kaski, and J. Kertész, Physical Review E, 75 027105 (2007). http://jponnela.com/web_documents/a9.pdf

>>> print(nx.clustering(G,0))
1.0
